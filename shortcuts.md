#### VS code 
1. markdown preview  
`(cmd + k) + v`

2. control palette  
`cmd + shift + p`

#### vim NERDTree
1. show/hide tree   
`fcn + F1 (customized)`
2. toggle between tree and vim window  
`ctrl + W + h/l` or `ctrl + W + </>`

#### tmux  
1. scroll mode in tmux
`ctrl + b + [` and `q` to quit scrolling mode
